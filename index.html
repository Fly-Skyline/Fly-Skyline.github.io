OpenCV-Dnn简介(详情见GitHub)

OpenCV3.3已经将深度学习DNN模块从opencv_contrib中移到了OpenCV正式发布模块中，当前DNN模块最早来自Tiny-dnn，可以加载预先训练好的Caffe模型数据，OpenCV做了近一步扩展支持所有主流的深度学习框架训练生成与导出模型数据加载，常见的有如下：
Caffe
TensorFlow
Torch/PyTorch
OpenCV中DNN模块已经支持与测试过这些常见的网络模块：
AlexNet
GoogLeNet v1 (also referred to as Inception-5h)
ResNet-34/50/…
SqueezeNet v1.1
VGG-based FCN (semantical segmentation network)
ENet (lightweight semantical segmentation network)
VGG-based SSD (object detection network)
MobileNet-based SSD (light-weight object detection network)
OpenCV通过支持加载这些预先训练好的模型，实现图像分类、对象检测、语义分割、风格迁移等功能。关于如何调用，OpenCV官方提供了GoogLeNet的示例：https://docs.opencv.org/trunk/d5/de7/tutorial_dnn_googlenet.html。
就在圣诞节前夕，OpenCV发布了最新的3.4版本，其中Dnn模块增加了R-CNN，详情见GitHub。

    std::pair::second // 概率百分比

  3. std::vector Classify(const std::vector &imgs);
    在1的基础上改进为批量图像数据处理。
  4. float Classify(const char filelabel, const char filerst = “”, char separator = ‘,’);
    图像数据分类批处理，filelabel为图像数据标签文件（separator指定其分割符），filerst为识别结果csv文件。

    filerst文件，每行结果格式示例：original/1/ALK01026_OBJECT.jpg,1,1,0.99834
